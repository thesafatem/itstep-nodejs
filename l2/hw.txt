Совместим знания об EventEmitter и файловой системе.
1. Создать переменную, которая будет в себе хранить путь до текущей папки (изначально пустая строка)
2. При совершении события "mkdir" создать папку с заданным именем в текущей папке
3. При совершении события "touch" создать файл с заданным именем и текстом в текущей папке
4. При совершении события "read" вывести на экран текст заданного файла
5. При совершении события "cd" перейти в указанную папку (обновить путь до текущей папки)
6. При совершении события "rm" удалить указанный файл, или папку
7. Эмитнуть события в разном порядке, чтобы проверить правильность написанного кода

Пример:
emit("mkdir", "folder1")
emit("cd", "folder1")
emit("mkdir", "folder11")
emit("cd", "folder11")
emit("touch", "hello.txt", "Hello, world!")
emit("cd", "..")
emit("cd", "..")
emit("mkdir", "folder2")
emit("touch", "salem.txt", "Salem, alem!")

В конце структура файлов и папок должна быть следующей:
root: 					// корневая папка с файлом задания (у вас может называться не root, а по-другому)
	folder1:
		folder11:
			hello.txt // с текстом Hello, world!
	folder2: 			// пустая папка
	index.js 			// файл с заданием
	salem.txt 		// с текстом Salem, alem!
