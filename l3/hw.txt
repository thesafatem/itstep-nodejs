1. Создать сервер на порту 3000
2. Если запрос пришел с методом POST:
записать студента из URL запроса в массив
3. Если запрос пришел с методом GET:
отправить в запросе имя студента с индексом из URL запроса
если такого индекса нет, то вернуть 404 ошибку
4. Если запрос пришел с методом DELETE:
удалить из массива студента с именем из URL запроса

Со звездочкой (желательно, не обязательно)
Записывать студентов не в массив, а в файл, читать тоже из файла
