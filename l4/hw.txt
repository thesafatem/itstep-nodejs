1. Создать приложение с помощью express
2. Создать изначально пустой массив фильмов. Фильм это объект с полями:
	name (string, название)
	year (number, год выхода)
	director (string, режиссер)
3. Добавить роут для публикации фильма (отправлять данные в body в json формате)
4. Добавить роут для получения всех фильмов, добавить к нему queryparams:
	?name - ищет фильмы, в названии которых есть строка name
	?year - ищет фильмы, вышедшие в году year
5. Добавить роут для получения фильма по его индексу
6. Добавить роут для получения всех режиссеров (если есть несколько фильмов с одним режиссером, режиссера надо показать 1 раз)
7. Добавить роут для удаления фильма по индексу
8. Добавить роут для удаления всех фильмов по году (год указывается через queryparams (?))
9. Добавить middleware, который проверяет индекс. Если индекса нет, то 404 ошибка. Если есть, то пускает дальше. Добавить midlleware в роуты 5 и 7.
