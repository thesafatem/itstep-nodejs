1. Создать вручную массив фильмов. Каждый фильм - объект, состоящий из названия, года выхода, имени режиссера, списка имен актеров.
2. Добавить роут для получения списка всех фильмов. На страничке должны быть только названия фильмов. Если кликнуть по названию, мы должны перейти по ссылке на страничку с информацией о фильме.
3. Добавить роут для получения информации о конкретном фильме (по индексу). Нужно показать все данные о фильме.
4. На нашем мини-сайте должен быть navbar, сделанный с помощью partials.

5*. Со звездочкой. Добавить роут для получения всех фильмов, в которых играл конкретный актер. Пример роута: localhost:3000/actors?name=Cruise
